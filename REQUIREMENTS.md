# 自動クリックアプリケーション - 要件仕様書

## 概要
ゲーム自動化を目的としたWindows専用の自動クリックツール。
システムトレイに常駐し、ホットキーで操作可能。

## 基本情報

| 項目 | 内容 |
|------|------|
| プロジェクト名 | Auto Clicker |
| 対象OS | Windows 10/11 |
| 技術スタック | Electron + TypeScript |
| UI形式 | システムトレイ常駐 + 設定ウィンドウ |

## 機能要件

### 1. クリック機能
- [ ] **クリック種別**: 左クリック / 右クリック / 中クリック
- [ ] **クリックタイプ**: シングル / ダブル
- [ ] **クリック間隔**: ミリ秒単位で設定可能（最小: 10ms、デフォルト: 100ms）
- [ ] **クリック回数**: 無限ループ または 指定回数

### 2. 位置指定機能
- [ ] **現在位置モード**: マウスカーソルの現在位置でクリック
- [ ] **固定位置モード**: 指定した画面座標でクリック
- [ ] **位置ピッカー**: 画面上の位置をクリックして取得するツール

### 3. ホットキー機能
- [ ] **カスタマイズ可能**: ユーザーが任意のキーを設定可能
- [ ] **デフォルト設定**:
  - 開始/停止トグル: F6
  - 一時停止: F7（オプション）
- [ ] **グローバルホットキー**: 他アプリがフォーカス中でも動作

### 4. システムトレイ機能
- [ ] **トレイアイコン**: 動作状態を視覚的に表示（アイコン色変更など）
- [ ] **右クリックメニュー**:
  - 開始/停止
  - 設定を開く
  - 終了
- [ ] **ダブルクリック**: 設定ウィンドウを開く

### 5. 設定ウィンドウ
- [ ] **クリック設定タブ**: 種別、間隔、回数の設定
- [ ] **位置設定タブ**: モード選択、座標入力
- [ ] **ホットキー設定タブ**: キーのカスタマイズ
- [ ] **設定の永続化**: 次回起動時に設定を復元

## 非機能要件

### パフォーマンス
- クリック間隔の精度: ±5ms以内
- メモリ使用量: 100MB以下
- CPU使用率: アイドル時1%以下

### ユーザビリティ
- 初回起動時の使い方ガイド（オプション）
- シンプルで直感的なUI
- 日本語対応

### 配布
- 単一exeファイルでの配布（ポータブル版）
- または インストーラー版の提供

## 主要ライブラリ

| ライブラリ | 用途 |
|-----------|------|
| electron | アプリケーションフレームワーク |
| robotjs または @nut-tree/nut-js | マウス操作 |
| electron-store | 設定の永続化 |
| electron-builder | ビルド・パッケージング |

## 開発フェーズ

### Phase 1: 基盤構築
- Electronプロジェクトのセットアップ
- TypeScript設定
- 基本的なシステムトレイ表示

### Phase 2: コア機能
- マウスクリック機能の実装
- ホットキーによる開始/停止
- 基本的な間隔設定

### Phase 3: UI実装
- 設定ウィンドウの作成
- 位置ピッカー機能
- ホットキーカスタマイズUI

### Phase 4: 仕上げ
- 設定の永続化
- アイコン・デザイン調整
- ビルド・配布設定

- アンチチート機能により検出される可能性があります
